Project

Project membuat prototipe aplikasi belajar, untuk membantu temen saya di jurusan DKV. Aset - aset button, logo, dan background serta sebagian desain disiapkan oleh teman saya. Tugas saya untuk membuat aplikasi dengan seluruh fitur yang diperlukan, membuat UI dan layout, serta mengatur database.

Features

  • Login, register, dan logout
  • Edit profile
  • Upload image and video
  • View other users image and video post
